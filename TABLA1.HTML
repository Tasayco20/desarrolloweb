Etiquetas HTML

Recuerda que es importante entender la diferencia entre ellas.

ETIQUETA UL+LI

si listamos personas nominadas a los Oscars, dado que el orden no altera el significado,debemos usar ul.

  • David Verdaguer
  • Jesùs Castro

Etiqueta ol+li

En el caso de que estemos listando elementos donde el orden es importante, como por ejemplo la tabla de posiciones de la COPA AMERICA 2021,debemos usar ol.

  2. Perù
  3. Argentina
  4. Chile
